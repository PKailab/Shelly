**Files:**

**調査結果 (Agent 2)**:
- 原因: `cat /proc/net/tcp6` が libbash.so サブプロセスで exit=1 失敗。PATH / SELinux / LD_LIBRARY_PATH 複合
- 推奨修正: Kotlin/JNI で fopen 直読 (shell skip、SELinux は Zygote ドメインで許可済み)

- [ ] **Step 1: shelly-exec.c (既存の JNI ソース) に関数追加**

既存の JNI ファイルを探す:
```bash
find ~/Shelly/modules/terminal-emulator -name "*.c" -o -name "*.cpp" 2>&1
```

追加する関数:
```c
JNIEXPORT jstring JNICALL
Java_expo_modules_terminalemulator_ShellyJNI_readProcNetFile(
    JNIEnv *env, jclass clazz, jstring pathJ)
{
    const char *path = (*env)->GetStringUTFChars(env, pathJ, NULL);
    FILE *f = fopen(path, "r");
    if (!f) {
        (*env)->ReleaseStringUTFChars(env, pathJ, path);
        return (*env)->NewStringUTF(env, "");
    }
    char buf[65536];
    size_t n = fread(buf, 1, sizeof(buf) - 1, f);
    buf[n > 0 ? n : 0] = '\0';
    fclose(f);
    (*env)->ReleaseStringUTFChars(env, pathJ, path);
    return (*env)->NewStringUTF(env, buf);
}
```
